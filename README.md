# .NET MVC Demo web

This is a .NET MVC website built with ASP.NET core version 8. 

## Features: 
- CRUD operations on Category Table
- CRUD operation on Products Table
- Used N-Tier Architecture
- Followed Coding Practices like Repository Pattern
- Used Bootstrap CSS for styling

## Still Work In Progress

- Different account managements for admin and normal users
- Home page
